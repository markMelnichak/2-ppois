Меню, содержащее список доступных блюд.

Позволяет пользователю выбрать блюдо по номеру и запустить его приготовление.

void Menu::run ( )

Основной цикл работы меню.

Выводит список блюд, читает выбор пользователя и вызывает Dish::cook() для выбранного блюда. Выход — выбор пункта 0 или некорректный ввод.
